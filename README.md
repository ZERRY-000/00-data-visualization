## Getting Started

### Prerequisites
- [Anaconda](https://www.anaconda.com/download) or Miniconda
- Python 3.x

### Installation

**1. Clone the repository**
```bash
git clone https://github.com/ZERRY-000/00-data-visualization.git
cd 00-data-visualization
```

**2. Open Anaconda Prompt and create a new environment**
```bash
conda create -n dataviz python=3.11
conda activate dataviz
```

**3. Install the required libraries**
```bash
conda install jupyter notebook
pip install plotly pydeck
pip install statsmodels
```

**4. Launch Jupyter Notebook**
```bash
jupyter notebook
```
